# check_mahjong_hand

麻雀の手牌を入力すると、待ちを判定し、全アガリ形を列挙する Web アプリです。  
オールマイティ牌 (何の牌としても使える牌)・[銀河麻雀](https://majyan-item.com/galaxy/)の銀河牌 (字牌以外) にも対応しています。  
https://check-mahjong-hand.herokuapp.com/
